# DataAnalysis_2026
# Module 1: Data Manipulation and Cleaning

### Assignment Description
This assignment demonstrates basic data handling in R, including exporting a dataset, performing a summary analysis, and generating a visualization.

### Data Description
* **Source:** Built-in R "Iris" dataset.
* **Format:** CSV file (`my_data.csv`).
* **Variables:** Sepal Length, Sepal Width, Petal Length, Petal Width, Species.

### Technical Details
* **R Version:** 4.5.3
* **Procedures used:** `write.csv` for data export, `summary()` for statistical analysis, and `plot()` for visualization.

### Files in this folder:
1. `my_data.csv` - The source data file.
2. `script.R` - The R code used for analysis.
3. `iris_plot.png` - The resulting visualization.
